# Contributing to Genjutsu Dev Terminal

First off, thank you for considering contributing to the Genjutsu Dev Terminal!  
It's people like you that make open-source such a great way to learn, inspire, and build together.

All contributions, from bug reports to new features, are welcome and appreciated.

---

## How Can I Contribute?

### 🐛 Reporting Bugs

If you find a bug, please let us know!

1. Check the [Issues](https://github.com/NOTAM-bobk/Shinobi.genjustu/issues) tab to see if it has already been reported.
2. If not, open a new issue.
3. Provide a clear title and description. Include steps to reproduce the bug if possible.

### ✨ Suggesting Enhancements

Have a cool idea for a new feature or a UI improvement?

- Open a new issue in the [Issues](https://github.com/NOTAM-bobk/Shinobi.genjustu/issues) tab.
- Explain your idea and why it would be useful for the project.

### 💻 Submitting Pull Requests (Code Changes)

If you want to add code yourself, follow these steps:

1. **Fork** the repository to your own GitHub account.
2. **Clone** your fork to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Shinobi.genjustu.git
