# Genjutsu Dev Terminal

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![React](https://img.shields.io/badge/React-18-61dafb.svg)](https://reactjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.0-3178c6.svg)](https://www.typescriptlang.org/)
[![Vite](https://img.shields.io/badge/Vite-5.0-646cff.svg)](https://vitejs.dev/)

A sleek, Vercel/Cloudflare-styled web client built with React and TypeScript, incorporating light neo-brutalist UI design elements. It fetches and previews code, media, and tech updates from the Genjutsu social feed API.

![Genjutsu Dev Terminal Demo](https://via.placeholder.com/1200x600?text=Genjutsu+Dev+Terminal+Demo)

## ✨ Features

- **Modern Stack** — React 18, TypeScript, Vite, Tailwind CSS
- **Clean UI** — Vercel-inspired monochrome typography mixed with offset shadows and sharp 2px borders for a neo-brutalist developer aesthetic
- **Live Code Execution** — Run HTML, CSS, and JS directly inside the feed
- **Analytics Widget** — Auto-calculating side panel mapping code lines, engagement, and top contributors
